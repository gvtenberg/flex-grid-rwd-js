# Flex/Grid + RWD + Eventos (JS)

Este projeto demonstra a criação de uma página web responsiva utilizando **HTML5**, **CSS3** e **JavaScript**.  
Foram aplicados conceitos de **Flexbox** (no header), **CSS Grid** (nos cards e galeria) e **RWD – Responsive Web Design**, com padrões como *Column Drop* e *Off-canvas Navigation*.

O site inclui **animações sutis**, **modo escuro automático**, e **feedback visual** com pseudoclasses `:hover`, `:focus` e `:active`.

O **JavaScript** adiciona interatividade, validando o formulário, alternando o menu mobile, mudando o header no scroll e integrando envio via **WhatsApp**.

### Execução
Para visualizar o projeto:
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Redimensione a janela para observar os comportamentos responsivos e interativos
